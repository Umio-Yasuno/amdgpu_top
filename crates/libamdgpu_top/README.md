# libamdgpu\_top

The `libamdgpu_top` crate is a crate that targets only `amdgpu_top`, and there is no documentation or examples at all.  
Breaking changes may be introduced in version upgrades.  
If you want to add AMD GPU monitoring functionality to a Rust project, we strongly recommend [amdsmi](https://github.com/ROCm/rocm-systems/tree/develop/projects/amdsmi) .
